# AllAiNovel 项目规则

## 项目概述
**这是为你一个人开发的AI辅助写作应用**。没有商业化压力，不需要考虑其他用户，一切以你的使用习惯和需求为主。

核心原则：
- **能少输入就少输入** - 减少一切不必要的打字
- **一键生成所有** - 输入提示词，AI自动搞定书籍、角色、大纲、章节
- **模型切换秒切** - 随时随地切换AI模型，不用跳转
- **不满意就改** - AI结果可以直接编辑，也可以输入指令让AI重新调整
- **提示词很随意** - 支持自然语言描述，AI自动理解并结构化

---

## 你的使用场景

### 典型 workflow
```
1. 打开App → 点击"新建书籍"
2. 输入一段随意的描述：
   "《斗破苍穹》同人，主角古无涯，古族大少爷，
    37岁九星斗圣巅峰，哑巴瞎子，老婆洛璃，
    妹妹古薰儿12岁，第一章写他们日常..."
3. AI自动解析生成：
   - 书籍信息
   - 角色档案（古无涯、洛璃、古薰儿...）
   - 世界观设定（古族、洛神族、斗气体系...）
   - 大纲结构
   - 第一章内容
4. 浏览生成结果 → 不满意的地方直接改或让AI重生成
5. 保存 → 开始写作
```

**入口**：书籍列表页 → "一键生成"按钮

**界面**：

│  │37岁九星斗圣巅峰，哑巴    │   │
│  │老婆洛璃，妹妹古薰儿...   │   │
│  │                          │   │
│  │第一章写日常...           │   │
│  └─────────────────────────┘   │
│                                 │
│  [生成选项]                      │

│                                 │
│  [模型: DeepSeek-v3 ▼]          │
│                                 │

│  └─────────────────────────┘   │

**AI解析逻辑**：
1. 提取书籍信息（标题、类型、简介）

5. 生成第一章内容

### 3. AI结果可编辑 + 修改指令

**生成结果卡片**：
```
┌────────────────────────────────────────┐
│ 生成结果                    [✏️编辑中] │
├────────────────────────────────────────┤
│ ┌────────────────────────────────────┐ │
│ │ 古无涯盘坐在古界深处的修炼室中...  │ │
│ │ 双目虽不能视，但感知却敏锐无比...  │ │
│ │                                    │ │
│ │ [可以直接在这里修改内容]           │ │
│ └────────────────────────────────────┘ │
├────────────────────────────────────────┤
│ 修改指令：                             │
│ ┌────────────────────────────────────┐ │
│ │ 语气更悲伤一些，增加环境描写       │ │
│ └────────────────────────────────────┘ │
│                                        │
│ [应用内容] [按指令修改] [重新生成] [清除]│
└────────────────────────────────────────┘
```

**操作说明**：
- **直接编辑**：在文本框里随便改
- **按指令修改**：输入修改要求，AI基于当前内容重新生成
- **重新生成**：完全重新生成新内容
- **应用内容**：保存到编辑器
